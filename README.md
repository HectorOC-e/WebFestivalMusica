# WebFestivalMusica

Esta web es una practica de SASS, GULP, JS y HTML.

### Requerimientos
##### Tener instalado NPM, Chocolatey, NodeJS
##### Ejecutar en la terminal
                npm i
##### Despues de haber instalado las dependencias ejecutar el siguiente comando
                npx gulp dev
